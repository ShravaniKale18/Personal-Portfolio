A modern, professional portfolio website built with HTML, CSS, and vanilla JavaScript. This project features a sleek, responsive design with smooth scrolling, interactive elements, and a glowing neon theme.

## Features

- ✨ **Modern Design**: Dark theme with neon accents and glassmorphism effects.
- 🖱️ **Smooth Scrolling**: Native JavaScript for smooth page navigation.
- 🎯 **Responsive Layout**: Fully adaptive design for desktop, tablet, and mobile devices.
- 📄 **Project Showcase**: Dedicated section to display projects with images and links.
- 📝 **Contact Form**: Easy way to get in touch via email.
- 💡 **Interactive Elements**: Hover effects, button animations, and scroll reveal effects.
- 🚀 **Performance Optimized**: Clean, minimal code for fast loading times.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge).
- [Node.js](https://nodejs.org/) (Optional, for development tools).
- [Git](https://git-scm.com/) (Optional, for version control).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ShravaniKale18/Personal-Portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Personal-Portfolio
   ```

3. **Open the `index.html` file** in your web browser:
   ```bash
   open index.html  # macOS
   # or
   start index.html # Windows
   ```

## Usage

Once the website is open in your browser, you can:

- Navigate through the sections using the top navigation bar.
- Click on the "View Projects" button to see my work.
- Use the contact form to send me a message.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Styling, animations, and responsiveness.
- **JavaScript (Vanilla)**: Scroll effects, navigation, and interactive features.
